# Stamen

> A sexy state container for React

`Stamen` 是一个 immutable React 状态管理库。

**如果**...

你也像我一样，不喜欢这些东西, 什么 provider, connections, actions, reducers, effects, dispatch, put, call, payload, @observable, @computed, @observer, @inject... 

你也像我一样，喜欢一堆模板代码、各种概念、什么大哲学... 还有各种多余牛毛的 Api。

你也像我一样，只是想早点下班。

你可以试试 Mob... 搞错了，是可以试试 `stamen`。

**为什么？**

- **轻量** gzip 后不到 700B，源码只有40行
- **简洁** 没模板代码, 仅有三个API
- **符合直觉** 没复杂的概念，只有 state 和 action
- **清晰** 更易写出可维护和可读性好的代码
- **高效** 更高的开发效率，这很重要
- **Typescript** 高度支持 state 和 action 的智能提示
- **Sexy** 能让你早点下班去找你的妹子
