# 快速上手

欢迎使用 Dahlia，在本指南，你将1分钟，学会使用 Dahlia CLI 构建并运行一个简单的 Dahlia 应用。

## 第一步: 安装 Dahlia CLI

推荐使用 yarn 安装

```bash
yarn global add dahlia
```

国内用户可以用淘宝镜像，先设置 alias:

```bash
alias tyarn="yarn --registry https://registry.npm.taobao.org"
yarn global add dahlia
```


## 第二步: 初始化应用

```bash
dh new my-app
```

## 第三步: 启动开发服务器

```bash
dh dev
```
